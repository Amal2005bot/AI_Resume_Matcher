# AI Resume Matcher
A web-based tool that ranks multiple resumes against a job description using **Natural Language Processing (NLP)**.

## Features
- **Multi-PDF Upload:** Analyze entire batches of resumes at once.
- **NLP Powered:** Uses spaCy for text cleaning and lemmatization.
- **Match Scoring:** Calculates similarity using TF-IDF and Cosine Similarity.
- **Exportable:** Download your candidate rankings as a CSV.

## Tech Stack
- Python, Streamlit, Pandas, Scikit-learn, spaCy
